# Notas clase 4

## Elementos de maquetado semántico

> Maquetar significa pasar a código HTML y CSS el contenido de nuestro prototipo/diseño

> En HTML tenemos ciertos elementos que fueron creados para poder dar estructura a nuestras páginas. Fueron creados para maquetar
> Algunos de estos elementos son:

    <header>   
        <nav>   
    <main>  
        <section>  
            <article> 
    <footer>  
    
## Box model (modelo de caja)
> Y no HTML todos los elementos son contenedores (cajas)
> por lo tanto podemos configurar atributos de caja
> podemos configurar el ancho, altura, color de fondo, bordes y también las separaciones (internas y externas)
